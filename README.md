# Things Eternal

A five-part interactive Bible study series exploring the topics that matter for eternity: God, the Bible, man, Jesus Christ, and salvation.

## Live Site

**[https://timjonez.github.io/things-eternal/](https://timjonez.github.io/things-eternal/)**

## Overview

“Things Eternal” is a self-paced, interactive Bible study built with plain HTML, CSS, and JavaScript. Each lesson includes fill-in-the-blank exercises, inline Bible verse popovers, and guided topical studies to help readers engage with Scripture directly.

## Lessons

1. **The Eternal Supreme Being (God)** — Who God is and what He is like
2. **The Eternal Scriptures (The Bible)** — The credibility and power of God’s Word
3. **The Everlasting Soul (Man)** — Creation, purpose, fall, and the problem of sin
4. **The Eternal Son (Jesus Christ)** — His life, death, and resurrection
5. **Eternal Salvation** — The meaning, effect, and way to salvation

## Features

- Interactive fill-in-the-blank exercises
- Inline Bible verse popovers on hover
- Scripture references quoted in full throughout the lessons
- Responsive, readable design
- No build step or dependencies — just open `index.html`

## Running Locally

Clone the repository and open `index.html` in any modern web browser:

```bash
git clone https://github.com/timjonez/things-eternal.git
cd things-eternal
open index.html
```
